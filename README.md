# API Backend Blog

Une API RESTful pour la gestion d'un blog, développée avec Node.js, Express et SQLite.

## Fonctionnalités
- CRUD complet des articles (Créer, Lire, Modifier, Supprimer).
- Recherche d'articles par mot-clé (titre ou contenu).
- Documentation interactive avec Swagger.

## Installation
1. Cloner le dépôt : `git clone <lien-du-repo>`
2. Installer les dépendances : `npm install`
3. Lancer le serveur : `npm run dev`

## Documentation
Une fois le serveur lancé, la documentation Swagger est disponible sur :
`http://localhost:3000/api-docs`
